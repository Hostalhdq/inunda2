Protocolo de Prevención y Teléfonos Útiles

Acciones preventivas ante eventos hidrometeorológicos severos en Concepción

132 Bomberos Concepción 133 Carabineros 131 SAMU Salud Emergencias Mun. Concepción
Antes de las Lluvias
  • Limpiar canaletas y rejillas de evacuación en su propiedad.
  • Retirar hojas y desechos acumulados en los sumideros de su calle.
  • Verificar sellado de puertas frente a posibles anegamientos.
Durante la Lluvia Intensa
  • Evitar transitar a pie o en vehículo por el paso bajo nivel UBB.
  • Mover vehículos a zonas con mayor cota de elevación si sube el estero.
  • Sintonizar canales oficiales de SENAPRED y Municipalidad.
En caso de Desborde
  • Desconectar suministros de energía eléctrica a nivel de piso.
  • Evacuar hacia sectores altos (hacia sectores consolidados del Collao).
  • Notificar en la app la situación exacta para guiar socorristas.

Guía Técnica: Lo que falta para Puesta en Producción 100%

1 Conectar Base de Datos Supabase (PostGIS)

Para persistir los reportes vecinales en tiempo real entre todos los usuarios, se requiere crear la tabla en Supabase: 

CREATE TABLE reportes_collao (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  latitud FLOAT NOT NULL,
  longitud FLOAT NOT NULL,
  categoria TEXT NOT NULL,
  nivel_severidad TEXT NOT NULL,
  descripcion TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  confirmaciones INT DEFAULT 1
);
2 Sensor IoT / API Estero Nonguén (DGA Chile)

Para reemplazar la cota simulada del estero por medición real, se puede conectar un sensor ultrasónico LoRaWAN o un Webhook de la DGA: 

// Endpoint HTTP Webhook para Sensor de Nivel
POST /api/v1/telemetria/estero-nonguen
Headers: { "X-Sensor-Key": "SECURE_KEY" }
Payload: { "nivel_metros": 0.85, "bateria": 98 }
